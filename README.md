# SunCart - Summer Essentials Store


## Purpose
A modern, responsive summer eCommerce platform SPA where users can explore and purchase seasonal products like sunglasses, summer outfits, skincare, and beach accessories.

## Key Features
- **Responsive Layout:** Works flawlessly across mobile, tablet, and desktop devices.
- **Product Listing & Viewing:** Static JSON listing for summer products with dedicated details pages.
- **Authentication:** Simulated login, registration, and Google OAuth implementation allowing users to authenticate seamlessly.
- **Protected Routes:** Certain pages (like Product Details and Profile) are only accessible to logged-in users.
- **Profile Management:** Users can view their profile and update their name and photo URL.
- **Animations:** Smooth transitions using Framer Motion and Animate.css for delightful user interactions.

## Technologies and npm Packages Used
- **React (Vite):** Core frontend framework for the Single Page Application.
- **Tailwind CSS & DaisyUI:** Used for utility-first styling and robust, accessible UI components.
- **React Router DOM:** Used to manage routing and protected routes within the SPA.
- **Animate.css & Motion (Framer Motion):** To fulfill the bonus challenge and provide beautiful entry/exit animations.
- **Lucide React:** Beautiful and consistent iconography.
- **React Toastify:** For elegant toast notifications across the application.
